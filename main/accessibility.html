Accessibility Statement

Last updated: 2026-02-25

Commitment

This application is designed with accessibility in mind to ensure that all users, including those with disabilities, can access essential features and information.

Supported Features

  • Readable text with clear structure
  • Consistent navigation across all pages
  • High‑contrast layout compatibility
  • Screen‑reader friendly HTML markup

Limitations

Some advanced features may not be fully optimized for assistive technologies. Updates may improve compatibility over time.

User Feedback

If you encounter accessibility issues, please report them through the Support page so improvements can be made.

Ongoing Improvements

Accessibility updates may be applied periodically to align with evolving standards and user needs.
