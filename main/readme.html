Application Documentation

Overview

This application is a web-based interface packaged for mobile use. All core files, legal documents, and supporting assets are located inside the main directory.

File Structure

The following files are included in the main directory:

  • index.html – Main application interface
  • privacy.html – Privacy Policy
  • terms.html – Terms of Service
  • billing.html – Billing & Subscription Terms
  • payouts.html – Payout Terms
  • license.html – Software License

Legal Compliance

All required legal documents are included directly in the main directory to ensure compliance with distribution requirements and platform policies.

Usage

Open index.html to launch the application. All legal pages can be accessed directly via their respective filenames.
